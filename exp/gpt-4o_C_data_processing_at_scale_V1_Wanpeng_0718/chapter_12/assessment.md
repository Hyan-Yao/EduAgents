**Question 1:** What is a common data issue that may arise during the project?

  A) Overfitting
  B) Missing values
  C) Lack of data
  D) Syntax errors

**Correct Answer:** B
**Explanation:** Missing values are a frequent challenge that data teams face during project work.

**Question 2:** Which method can be used to identify outliers in a dataset?

  A) Mean substitution
  B) Box plots
  C) Normalization
  D) Data encoding

**Correct Answer:** B
**Explanation:** Box plots are effective visual tools for detecting outliers within a dataset.

**Question 3:** What technique can be utilized to handle missing values besides deletion?

  A) Variable transformation
  B) Data encoding
  C) Imputation
  D) Feature selection

**Correct Answer:** C
**Explanation:** Imputation involves replacing missing values with statistical estimates, such as mean or median.
