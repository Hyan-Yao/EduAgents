## Section 7: Utilizing Evaluation Metrics

### Learning Objectives
- Understand various evaluation metrics used in assessing model performance.
- Apply precision, recall, and F1 score calculations in practical scenarios.

### Assessment Questions

**Question 1:** What does precision measure in model evaluation?

  A) The accuracy of predictions made by the model
  B) The ratio of correctly predicted positive observations to the total predicted positives
  C) The overall performance of the model
  D) None of the above

**Correct Answer:** B
**Explanation:** Precision is specifically about the accuracy of positive predictions.

**Question 2:** In a scenario with a highly imbalanced dataset, which evaluation metric should you be cautious of relying solely on?

  A) Precision
  B) Recall
  C) Accuracy
  D) F1 Score

**Correct Answer:** C
**Explanation:** Accuracy may give a misleading sense of performance when one class is significantly larger than the other.

**Question 3:** What is the F1 Score used for in model evaluation?

  A) To measure the accuracy of the model
  B) To provide a balance between precision and recall
  C) To measure the number of false positives only
  D) To determine overall model complexity

**Correct Answer:** B
**Explanation:** The F1 Score combines precision and recall into a single metric, useful for balancing both in model evaluation.

**Question 4:** If a model has a recall of 0.85, what does this imply about its performance?

  A) It has correctly identified 85% of all positive cases.
  B) It predicts 85% of all predictions correctly.
  C) It has high precision.
  D) It indicates the total number of predictions made.

**Correct Answer:** A
**Explanation:** A recall of 0.85 indicates that the model has correctly identified 85% of the actual positive cases.

### Activities
- Given a confusion matrix with values: True Positives = 30, False Positives = 15, True Negatives = 50, False Negatives = 5, calculate precision, recall, and F1 Score.

### Discussion Questions
- In what scenarios might you prioritize recall over precision, and vice versa?
- How can you mitigate the risks associated with relying on a single evaluation metric in model assessment?
